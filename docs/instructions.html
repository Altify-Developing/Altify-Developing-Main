Simple Site - Technical Details and Documentation

The pages included below this one contain further technical details of how the Simple Site system works. The two key files pages.json and site.json, used to control and define the new webpages, are described in detail, along with the various extensions to Simple Site, that have been created to add more complex functionality, like adding IIIF viewers.

The other set of template pages have been included to provide some more simple examples of how "pages" can be defined. A copy of these page have also been included as the pages-template.json file found in the build folder in your Github Repository.
